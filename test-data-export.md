# 数据导出功能测试指南

## 测试步骤

### 1. 访问数据导出页面
1. 登录系统（使用管理员账号：admin/admin123）
2. 在左侧菜单中找到"统计分析" -> "数据导出"
3. 点击进入数据导出页面

### 2. 测试学生信息导出
1. 选择"学生信息"数据类型
2. 设置筛选条件：
   - 班级：可选，例如"计算机2401"
   - 搜索：可选，例如"张三"
3. 选择导出格式：JSON/CSV/Excel
4. 点击"预览数据"按钮
5. 确认预览数据正确显示
6. 点击"导出数据"按钮
7. 检查下载的文件是否正确

### 3. 测试课程信息导出
1. 选择"课程信息"数据类型
2. 设置筛选条件：
   - 搜索：可选，例如"日语"
3. 选择导出格式：JSON/CSV/Excel
4. 点击"预览数据"按钮
5. 确认预览数据正确显示
6. 点击"导出数据"按钮
7. 检查下载的文件是否正确

### 4. 测试学生成绩导出
1. 选择"学生成绩"数据类型
2. 设置筛选条件：
   - 班级：可选，例如"计算机2401"
   - 课程：可选，选择"日语"
   - 学号：可选，例如"2024001"
   - 时间范围：可选，选择日期范围
3. 选择导出格式：JSON/CSV/Excel
4. 点击"预览数据"按钮
5. 确认预览数据正确显示（包含学生班级信息）
6. 点击"导出数据"按钮
7. 检查下载的文件是否正确

### 5. 测试筛选功能
1. 测试班级筛选：输入"计算机2401"，只应显示该班级的数据
2. 测试课程筛选：选择特定课程，只应显示该课程的成绩
3. 测试学号筛选：输入特定学号，只应显示该学生的成绩
4. 测试时间范围：选择日期范围，只应显示该时间段内的成绩
5. 测试重置按钮：应清空所有筛选条件

### 6. 测试格式转换
1. JSON格式：应包含完整的数据结构
2. CSV格式：应为表格格式，可用Excel打开
3. Excel格式：应为可直接用Excel打开的格式

### 7. 测试权限控制
1. 学生角色：只能导出自己的成绩数据
2. 教师/管理员角色：可以导出所有数据

## 预期结果

✅ 筛选条件正确应用
✅ 数据预览准确显示
✅ 文件下载正常

✅ 权限控制有效
