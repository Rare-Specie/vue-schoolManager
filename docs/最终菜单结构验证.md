## 用户体验

### 教师/管理员用户
1. 登录后，左侧导航栏显示"统计分析"子菜单
2. 点击可展开看到"统计概览"和"数据导出 (待开发)"
3. 点击"统计概览"进入统计概览页面
4. 点击"数据导出 (待开发)"进入占位页面，看到功能说明

### 学生用户
1. 登录后，左侧导航栏显示单个"统计概览"菜单项
2. 点击直接进入统计概览页面
